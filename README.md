# CyberMeow
Nuestro Espacio es una aplicación web full-stack que combina desarrollo backend con Flask y renderizado 3D en Three.js para crear una experiencia social virtual. El proyecto incluye autenticación de usuarios, personalización de avatares, gestión de perfiles y navegación en entornos tridimensionales interactivos.

#  Nuestro Espacio

hola a **Nuestro Espacio**, un proyecto personal inspirado en juegos sociales como Habbo Hotel, pero con una temática de gatos.

La idea es crear un pequeño mundo virtual donde los usuarios puedan registrarse, personalizar un gatico y recorrer diferentes espacios mientras interactúan con otros usuarios.

Este proyecto nació principalmente como una forma de aprender y experimentar con tecnologías web, modelado de entornos 3D y desarrollo full-stack. No pretende ser una copia de Habbo ni una plataforma comercial; es simplemente un proyecto de hobby desarrollado en mi tiempo libre.

---

## Características actuales

* Registro e inicio de sesión de usuarios.
* Sistema de perfiles.
* Creación y personalización de gaticos.
* Entorno virtual renderizado con Three.js.
* Navegación dentro del mapa.
* Backend desarrollado con Flask.
* Persistencia de datos mediante base de datos relacional.
* Arquitectura modular para facilitar futuras expansiones.

##  Estado del proyecto

El proyecto se encuentra en desarrollo activo.

Actualmente una de las principales dificultades es la sincronización en tiempo real del movimiento de los gaticos entre usuarios conectados. Existen comportamientos que todavía no son completamente fluidos o consistentes cuando varios jugadores interactúan simultáneamente.

Debido a esto, algunas funcionalidades multijugador aún pueden presentar errores visuales o de sincronización.
